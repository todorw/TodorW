## About

I'm a security-focused software engineer based in **Montenegro**, working across the stack — from bare-metal systems and kernel internals to backend architecture and web applications. My current focus is **offensive and defensive security**: vulnerability research, exploit development, network security, and the low-level systems knowledge that makes both possible.

I care less about shipping fast and more about shipping *correct* — code and systems that hold up under scrutiny, not just under happy-path testing.

## Projects

### 🔩 [KumOS](https://github.com/TodorW/KumOS)
> *A hobby operating system written from scratch in C.*

Custom bootloader, kernel design, memory management, and x86 architecture deep-dives. The kind of project you build when you want to know what software is *actually* made of — no abstractions, no shortcuts. Direct groundwork for the low-level systems knowledge that security research depends on.

`C` · `x86 Assembly` · `Bare Metal` · `OS Design`
